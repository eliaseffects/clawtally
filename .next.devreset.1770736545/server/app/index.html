Read-Only Safety

ClawBoard does not send tasks to your agent. It only reads usage telemetry for dashboards and leaderboards.

Safety and Scope

Your gateway stays in control. ClawBoard only reads stats.

  • Read-only telemetry only (`agents.list`, `sessions.usage`).
  • No command execution, no tool invocation, no file changes.
  • No remote control over your OpenClaw instance.
  • Sharing is explicit opt-in and anonymous by default.
